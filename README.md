# rtos-scheduling


<!-- ABOUT THE PROJECT -->
## About The Project
RTOS scheduler simulation that implements the following scheduling algorithms:
- Round Robin
- Priority Based
- Modulo Based Round Robin
- Modified Round Robin
- Priority Based Round Robin

<!-- USAGE EXAMPLES -->
### Installation and Run


-> Build
   ```sh
   mkdir build
   make
   ```

-> Run
   ```sh
   ./scheduler
   ```
